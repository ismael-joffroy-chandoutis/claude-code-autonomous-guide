# Claude Code as an Autonomous Production Machine

Dynamic workflows, loops and goals: what is verified, what is folklore, and how to actually build the machine.

**→ [Read the guide in English](GUIDE.en.md)**

## Method

This guide was researched with the method it describes: a Claude Code dynamic workflow ran 109 agents across 26 sources, extracted 125 claims, and adversarially verified the top 25 with three independent verifier votes each (23 confirmed, 2 refuted, including the viral claim that an official `/goal` command exists: it does not).

## Status

Dynamic workflows are a **research preview** (June 2026). Names, caps and keywords may change. Issues and corrections welcome.
