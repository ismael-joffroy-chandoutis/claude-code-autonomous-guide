## Method / Méthode

This guide was researched with the method it describes: a Claude Code dynamic workflow ran 109 agents across 26 sources, extracted 125 claims, and adversarially verified the top 25 with three independent verifier votes each (23 confirmed, 2 refuted, including the viral claim that an official `/goal` command exists: it does not).

Ce guide a été documenté avec la méthode qu'il décrit : un dynamic workflow de Claude Code a lancé 109 agents sur 26 sources, extrait 125 claims, et vérifié les 25 principaux par trois votes adversariaux indépendants chacun (23 confirmés, 2 réfutés, dont le claim viral de l'existence d'une commande `/goal` officielle : elle n'existe pas).
