# Claude Code comme machine autonome de production

Dynamic workflows, loops et goals : ce qui est vérifié, ce qui est du folklore, et comment construire réellement la machine.

**→ [Lire le guide en français](GUIDE.fr.md)**

## Méthode

Ce guide a été documenté avec la méthode qu'il décrit : un dynamic workflow de Claude Code a lancé 109 agents sur 26 sources, extrait 125 claims, et vérifié les 25 principaux par trois votes adversariaux indépendants chacun (23 confirmés, 2 réfutés, dont le claim viral de l'existence d'une commande `/goal` officielle : elle n'existe pas).

## Statut

Les dynamic workflows sont en **research preview** (juin 2026). Noms, plafonds et mots-clés peuvent changer. Issues et corrections bienvenues.
